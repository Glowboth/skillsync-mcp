## 🛠️ How to Use skillsync-mcp

### Launch the App

- Find the skillsync-mcp icon on your desktop or in the Start menu.
- Double-click to open it.

### Basic Interface Overview

- The main window shows a list of AI skills currently added.
- You can scan new skills by clicking the **Add Skill** button.
- When you add a skill, the app runs a security check automatically.
- If a threat is found, the app shows a warning and blocks the skill.
- You can remove or update skills anytime using the buttons in the list.

### Scanning Skills

- Click **Scan Now** to check all added skills for new threats.
- The results will appear in the window. Safe skills will have a green mark. Skills with issues show a red mark.
- You can click on any skill to see details about the threats or risks found.
